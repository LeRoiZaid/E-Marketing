# 📱 E-Markting App
</p>

**E-Markting** is a mobile application built with **Flutter** that allows users to publish products, set prices, browse listings, and make purchases seamlessly. The app focuses on providing a smooth user experience enhanced with **animations** for engaging navigation and interactions.

---

## ✨ Features
- 🛒 **Product Publishing**: Users can add new products with details, images, and prices.
- 💰 **Flexible Pricing**: Sellers can set product prices according to their preferences.
- 🔍 **Product Browsing**: Clean and organized interface for exploring available items.
- ⚡ **Direct Purchase**: Simple and intuitive flow for buying products.
- 🎞️ **Animations**: Smooth transitions and visual effects to enhance usability and engagement.

---

## 🛠️ Tech Stack
- **Flutter**: Cross-platform framework for Android, iOS, Web, and Desktop.
- **Dart**: Core programming language.
- **Assets & Images**: Product image management.
- **Animations**: Custom transitions and effects for better UX.

